# Reference Materials

This folder contains reference materials and documentation used by the AWS Audit Agent System.

## Contents

### isaca-audit-programs/

Contains the ISACA AWS Audit Program CSV files (2022 version) that define the control objectives and testing procedures for AWS audits.

These files serve as the foundation for:
- Control domain definitions
- Audit testing procedures
- Compliance requirements
- Modernization recommendations

See `isaca-audit-programs/README.md` for detailed information about each file.

## Purpose

The reference materials in this folder provide:

1. **Audit Standards** - ISACA control frameworks and procedures
2. **Control Objectives** - What the audit aims to validate
3. **Testing Procedures** - How controls should be tested
4. **Compliance Mappings** - Alignment with industry standards

## Usage

The audit agents will reference these materials to:
- Understand control objectives
- Execute appropriate testing procedures
- Generate findings aligned with ISACA standards
- Produce modernization recommendations
