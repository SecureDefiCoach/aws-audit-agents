### The Audit Team (7 Agents)

**Management**
- **Maurice** - Audit Manager (GPT-4 Turbo)

**Senior Auditors**
- **Esther** - IAM & Logical Access (GPT-5)
- **Victor** - Logging & Monitoring (GPT-5)

**Staff Auditors**
- **Hillel** - IAM Support (GPT-4 Turbo)
- **Neil** - Encryption & Network Support (GPT-4 Turbo)
- **Juman** - Logging Support (GPT-4 Turbo)

**Company Representative**
- **Chuck** - CloudRetail IT Manager (GPT-4 Turbo)

### The 6 Audit Phases

1. **Risk Assessment & Planning** (2-3 weeks)
2. **Control Testing (Fieldwork)** (3-4 weeks)
3. **Workpaper Review & QA** (1-2 weeks)
4. **Remediation Planning** (1 week)
5. **Audit Reporting** (1-2 weeks)
6. **Follow-Up** (Optional)
